#Classification of NPR Interviews

####Setup

```
pip install -r requirements.txt
python -m nltk.downloader all
```

Note: You may need to run commands with root priveleges (sudo).

####Supported classifiers
- Bayes
- Decision Trees
- K nearest neighbor
